    增加index.js文件

    #!/usr/bin/env node必不可少，告诉操作系统，使用node解析该文件

    此时执行node index.js命令可以执行index.js文件
    那么，如何执行lv命令就能执行index.js文件呢？
    首先添加bin之后就可以了，但是我们现在在本地，我们的文件还没有发布至npm，本地也未安装，便没有这个命令，但是我们可以模拟

    npm link

    在本地开发npm模块的时候，我们可以使用npm link命令，将npm模块链接到对应的运行项目中去，方便地对模块进行调试和测试

    显然，运行lv命令执行了index.js文件
    这意味着当我们使用npm安装该脚手架后，便能使用lv命令了
