    迭代器

    可以被next()函数调用并不断返回下一个值的对象称为迭代器：Iterator

    凡是可作用于for循环的对象都是可迭代对象
    凡是可作用于next()函数的对象都是迭代器，它们表示一个惰性计算的序列
    集合数据类型如list、dict、str等是可迭代对象但不是迭代器
    可以通过iter()函数获得一个Iterator对象

    判断是迭代器或可迭代对象

    isinstance判断是否为指定类型
