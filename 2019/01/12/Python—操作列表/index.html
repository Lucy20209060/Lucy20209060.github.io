    使用列表的一部分

    切片[start:end]

    包括start位置的元素，不包括end位置的元素
    start 切片起点，默认0
    end 切片终点，默认列表长度

    切片复制列表

    元组

    列表可以修改，适合用于存储在程序运行期间可能变化的数据集。
    不可变的列表称为元组，存储一组在程序的整个生命周期内都不变的列表
    元组使用()来标识，定义元组后，就可以使用索引访问其元素
