    os 模块提供了操作系统相关的实用方法

    os.EOL

    一个常量，返回当前操作系统的换行符（Windows系统是\r\n，其他系统是\n）

    os.arch()

    表明 Node.js 二进制编译所用的 操作系统CPU架构
    现在可能的值有: 'arm', 'arm64', 'ia32', 'mips', 'mipsel', 'ppc', 'ppc64', 's390', 's390x', 'x32', 'x64'

    os.cpus()

    返回一个对象数组, 包含每个逻辑 CPU 内核的信息

    os.freemem()

    以整数的形式回空闲系统内存的字节数

    os.homedir()

    以字符串的形式返回当前用户的home目录

    os.hostname()

    以字符串的形式返回操作系统的主机名

    os.platform()

    返回Node.js编译时的操作系统平台
    当前可能的值有: 'aix'，'darwin'，'freebsd'，'linux'，'openbsd'，'sunos'，'win32'

    os.tmpdir()

    返回操作系统的默认临时文件目录
