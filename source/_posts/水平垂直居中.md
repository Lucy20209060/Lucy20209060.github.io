---
title: 水平垂直居中
date: 2017-08-02 21:56:51
tags: css
summary: 水平垂直居中的六种方式
---


```css
/*方法一*/
div{
    position:absolute;
    width:100px;
    height:100px;
    top:calc(50% - 50px);
    left:calc(50% - 50px);
}

/*方法二*/
div{
    position:absolute;
    top:50%;
    left:50%;
    transform:translate(-50%,-50%);
}

/*方法三*/
div{
    width:100px;
    height:100px;
    position:absolute;
    top:0;
    bottom:0;
    left:0;
    right:0;
    margin:auto;
}

/*方法四*/
父元素{
    display:flex;
    align-item:center;
    justify-content:center;
    div{
        width:100px;
        height:100px;
    }
}

/*方法五*/
父元素{
    display:flex;
    div{
        width:100px;
        height:100px;
        margin:auto;
    }
}

/*方法六*/
父元素{
    display:table;
    div{
        display:table-cell;
        vertical-align:middle;
    }
}
```
