    Sequel Pro连接mySql

    Sequel Pro是一款管理 Mysql 的工具，界面简洁易用，你可以一次性连接多个数据库，允许快速访问那些你经常需要访问的数据库

    连接出现错误

    Sequel Pro连接本地数据库时，出现了错误MySQL said: Authentication plugin ‘caching_sha2_password’ cannot be loaded: dlopen(/usr/local/…

    错误显示，链接数据库时不能加载‘caching_sha2_password'这个插件，也就是不能对身份验证

    解决方法

    打开系统偏好设置，找到mysql

    点击Initialize Database，输入数据库密码，选择 Use Legacy Password Encryption

    ok，连接成功
